1. App Móvil para Trabajadores Sociales

Descargue aquí la aplicación movil para Android para el trabajo de campo para Trabajadores Sociales.

Esta aplicación le permite:

  • Gestionar su agenda
  • Examinar el Historial Social Único de sus Usuarios
  • Examinar el Historial Familiar
  • Cumplimentar informes de habitabilidad
  • Tomar notas, fotos in situ y localizar y registrar direcciones.

Pulse sobre el movil para ver las diferentes pantallas.

2. Catalogo Móvil para Ciudadanos

Con la siguiente aplicación movil puede acceder al catalogo de Servicios, Profesionales y Recursos. Seleccione la version adecuada a su dispositivo movil e instale directamente la aplicación.

Instale la aplicación en su teléfono o tableta Android a traves de los siguientes pasos:

  1. Necesitará autorizar la instalación de aplicaciones externas en su Android. Puede hacerlo en:
    • Para Android 4.0+: Configuración > Seguridad > Marque "Fuentes desconocidas"
    • Para Android 4.0: Configuración > Aplicaciones Marque "Fuentes desconocidas"
  2. Ahora pinche en este enlace desde su teléfono o tableta, o bien puede emplear siguiente codigo BIDI/QR.
